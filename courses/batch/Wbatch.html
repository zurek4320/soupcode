What is Batch & Hello, World!

A batch file is a script file in DOS, OS/2 and Microsoft Windows. It consists of a series of commands to be executed by the command-line interpreter, stored in a plain text file. A batch file may contain any command the interpreter accepts interactively and use constructs that enable conditional branching and looping within the batch file, such as IF, FOR, and GOTO labels. The term "batch" is from batch processing, meaning "non-interactive execution", though a batch file might not process a batch of multiple data. It can be used for writing promts in Windowses CMD.

Batch Hello, World!

Making a "Hello World" program in batch couldn't be easier, to print just write echo Text you want to print. So hello world would look like this:

echo Hello, World!

Well, if this script would be ran by itself, not inside of CMD it would end immediately making it imposible to read message. To prevent that we can use pause function. pause stops the program until any key is pressed. But then still output would be more something like this: 

C:\Users\zurek>echo Hello, World!
Hello, World!

C:\Users\zurek>pause
Press any key to continue . . .

To stop the program from writing output like it's a CMD we can turn @echo off. With all of that here is a complete Hello, World! program:

@echo off
echo Hello, World!
pause

Variables & Basic Arithmetic

In batch defining a string variable is really easy, write set followed by variable_name= and variable content without any double quotes. For example:

set name=bob

We can later print that variable by putting its name into %%, for example:

@echo off
set name=Bob
echo My name is %name%.
pause

Arithmetic

Making an int variable in batch is nearly the same as making a string one but before the variable_name we add /a like this:

set /a num=420

Adding, subtracting etc. is really easy, just make variable exual to X(+|-|*|/|)Y for example:

@echo off
set /a num=7
set /a num2=num+5
echo %num% + 5 = %num2%
pause

If Stagments & Input

In batch getting input is really easy, just add /p after set while declaring a variable, for example: set /p name=. If you want to write something at the same line before input will be given you can write it after =, for example here is "What's your name?" program:

@echo off
set /p name=What's your name? 
echo Hello, %name%!
pause

If Stagments

If stagments are a bit diffrent that what you might expect, variables need to be inside of %...% and if its string in "..." too (as well as the text you are compering it to). After if stagments there need to be (...) for example:

@echo off
echo Question: 2+2=?
set /p answer=
if %answer%==4 (
  echo You are right!
)
pause

Elif and Else

Batch doesn't really have elif, you need to use else if. else and else if need to be putted right after closing of an if stagment for example:

@echo off
echo Question: 2+2=?
set /p answer=
if %answer%==4 (
  echo You are right!
) else if %answer%==22 (
  echo Nope???
) else (
  echo You are wrong D:
)
pause

Labels

What is a label? Label is a location indicator that code can later move to. To define a label write :label_name and to go that label write goto :label. Here is an example of label usage:

@echo off
goto :end
echo hello there
:end
pause

Code above won't print hello there! because it skips echo hello there! by jumping to a label called end.

Ala Functions

In batch you can call a function by using call keyword. It will run the code inside of that function and then comeback to code from before. Function need to end with goto :eof which you shouldn't define or it won't work. Here is an example: 

@echo off
call :hello
goto :end
:hello
  echo hello there!
  goto :eof
:random_label
  echo This won't print!
:end
  pause

Cool Tricks

Here are some cool tricks you can use in your batch creations.

CLS

cls clears whole screen. Yea, that's all

Case Insensitive If Stagments

To make if stagment case insensitive add /i right after if.

!=, >=, <=, >, <

Well, batch doesn't support operators like <= or != but support numeric comparisons. Here is the table of them:

  • equ — Equal Example: if %x% equ %y%
  • neq — Not equal Example: if %x% neq %y%
  • lss — Less than Example: if %x% lss %y%
  • leq — Less or equal Example: if %x% leq %y%
  • gtr — Greater than Example: if %x% gtr %y%
  • geq — Greater or equal Example: if %x% geq %y%

Code example:

With all that knowlegde in mind, here is a little quiz game example:

:: This quiz is very short but if you want you can easly extend it

@echo off
echo Welcome to clever quiz!
pause
cls

echo Question 1.
echo What's 2+2?
echo a: 5  b: 22
echo c: 4  d: 2

set /p answer=Answer (a/b/c/d): 
if /i "%answer%" neq "c" (goto :lose)

echo Correct!
pause
cls

echo Question 2.
echo What's the most clever code learning website?
echo a: SoupCode  b: Youtube
echo c: Google    d: HowToPayTaxes

set /p answer=Answer (a/b/c/d): 
if /i "%answer%" neq "a" (goto :lose)

echo Correct!
pause
cls

goto :win
:lose
  echo You have lost!
  goto :end
:win
  echo You won! (yes this quiz had only 2 questions)
:end
  pause
