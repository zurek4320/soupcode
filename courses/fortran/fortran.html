What is Fortran & Hello, World!

Fortran is a third-generation, compiled, imperative programming language that is especially suited to numeric computation and scientific computing. Fortran was originally developed by IBM with a reference manual being released in 1956 however, the first compilers only began to produce accurate code two years later. Fortran computer programs have been written to support scientific and engineering applications, such as numerical weather prediction, finite element analysis, computational fluid dynamics, plasma physics, geophysics, computational physics, crystallography and computational chemistry. It is a popular language for high-performance computing and is used for programs that benchmark and rank the world's fastest supercomputers.

Hello, World!

First of all to make a working program you need to write program program_name at the very beginning of the code and end program program_name at the end of it. Printing is very simple, you can do it in diffrent ways but we will stick with the most common one for now. To print write print followed by printing way specificator, a comma and text to print inside of "...". To make sure everything is working add implicit at the beginning of the program, it will be useful later but it's a good habit to always add it. So with all that in mind here is the hello world program:

program hello
  implicit none
  print *, "hello world!"
end program hello

String variables & Input

In Fortran defining variables might feel a little bit weird, to do it (define a variable) you need to type variable_type :: variable_name. First variable type you gonna learn is a string that in Fortran is called character. BUT, defining a string is a little bit diffrent than defining any other variable, you need to specify how much characters can your variable store at most. You can do that in few diffrent ways for example by writing characters limit after * at the end like this: character :: clever_var_name*20 or by writing it in brackets like this: character(20) :: first_name. Now that you defined a variable to make it hold information write variable_name followed by = and value of your variable (text values need to be inside of double quotes) for example: data = "clever text". To print a variable you can close double quotes in print and add your variable (or other text in another double quotes) after comma. Here is an example of that:

program string
    character :: name*20
    name = "zurek"
    print *, "Hello, ", name, "!"

end program string

After running code above you will see following output: Hello, zurek !. But why? That's because we specified that variable name can hold up to 20 characters, so the print prints 20 character. To delete all the spaces in print after the variable we can use function called trim. Here is an example of code that will give us this output: Hello, zurek!

program string
    character :: name*20
    name = "zurek"
    print *, "Hello, ", trim(name), "!"

end program string

Input

In Fortran getting input is really easy, just type read *, followed by name of variable you want to save it to. For example:

program test
	character :: data*50
	print *, "Enter data:"
	read *, data
	print *, "You have entered: ", data
end program test

Integer and Real Variables & Basic Arithmetic

Integer variables are these ones that hold, well integers and are defined like this: integer :: number. Real variables hold fractional numbers and are defined like this: real :: number. Here is an example usage of them:

program test
	integer :: num1
	real :: num2
	num1 = 48
	num2 = 3.14
	print *, "Num1: ", num1, "| Num2: ", num2

end program test

Well, real numbers are a little bit laggy so as the output you will get something more like this: Num1: 48 | Num2: 3.14000010 than this: Num1: 48 | Num2: 3.14000000
